Тренажёр слов на React

Описание проекта
Приложение предназначено для изучения иностранных слов. Пользователь может просматривать случайные слова и показывать их перевод. Проект создан на React с использованием функциональных компонентов и хуков.

Функциональные возможности:
- Вывод случайного слова из списка (более 100 слов).  
- Кнопка "Показать перевод" для отображения перевода слова.  
- Кнопка "Следующее слово" для перехода к следующему случайному слову.  
- Возможность расширения списка слов и функционала.

Техническая реализация:
- **React**: функциональные компоненты, хуки `useState`.  
- **Компоненты**: 
  - `App.js` — основной компонент, управляет состоянием и логикой отображения.  
  - `WordCard.js` — компонент карточки слова с кнопками для показа перевода и перехода к следующему слову.  
- **Стилизация** через CSS (`styles.css`).  
- **Сборка и запуск** через npm: `npm install` → `npm start`.  
- **Структура проекта**:
Скриншоты работы сайта, предоставлены в отдельной папке под названием "screnshoot"
